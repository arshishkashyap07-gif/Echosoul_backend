# EchoSoul Backend

AI-powered emotional journaling backend built with FastAPI.

## Features
- JWT-based authentication
- Secure journaling APIs
- Emotion analysis using NLP
- Production-ready backend structure

## Tech Stack
- FastAPI
- SQLAlchemy
- JWT Authentication
- Python NLP models
